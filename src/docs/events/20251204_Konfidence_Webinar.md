# Upcoming Webinar: Efficient Microservices Delivery with Konfidence

Konfidence, a prospective NeoNephos project, hosts a public webinar on **December 4th** at 3:00 pm CET.
Sarah Müller and Timon Schüle, both engineering manager SAP, dive into how Konfidence fits into the the NeoNephos project stack through its relation to the Apeiro Reference Architecture. You'll learn how it streamlines software releases, reduces infrastructure complexity, and supports modern deployment strategies like ring deployments and feature toggle management, all while following CNCF best practices.

-   **Speakers**: Sarah Müller, and Timon Schüle, Engineering Managers, SAP
-   **Host**: SAP Open Source Program Office
