# Welcome to the LF NeoNephos Website Repository 

:wave: Welcome to the GitHub presence of the [NeoNephos Foundation's website](https://neonephos.org).

## What is the purpose of this repository?

To collaborate on news articles, blog entries and other website content here.

## :globe_with_meridians: Table of Contents

- **[Blog](./content/blog/entries):** Files here configure the [Blog](https://neonephos.org/blog/).
- **[Events](./content/events/entries):** Files here configure the [Events page](https://neonephos.org/events/).
- **[Projects](./content/projects/overview/):** Files here configure the [Projects page](https://neonephos.org/projects/).

To facilitate an open and welcome environment for all, check out the [Linux Foundation Europe Code of Conduct](https://linuxfoundation.eu/en/policies?__hstc=81619592.11739e1f4ca03192f4e918409f725053.1748961872693.1753777412155.1753863796744.19&__hssc=81619592.3.1753863796744&__hsfp=2164450451).
## How to add content

If you are a TSC member, the following repositories are most relevant if you want to add content or change descriptions of the website.

- **[Events](src/docs/events)**: Add events (community calls, conventions etc.) here
- **[Projects](src/docs/projects.md)**: Change project descriptions here (you must *edit* this file in order to see the full content).
- **[Technical Advisory Council](src/docs/technical_advisory_council.md)**: Change descriptions and images of TAC members (you must *edit* this file in order to see the full content).
- **[Governing Board](src/docs/governing_board.md)**: Change descriptions and images of GB members (you must *edit* this file in order to see the full content).
- **[Blog](src/docs/blog)**: Add blog/news entries here.

Pull Requests are always welcome!
Check the issues page for outstanding issues.
